# Landing Page de Zapatos Ergonómicos de Bebé
¡Bienvenido/a al repositorio de la landing page de zapatos ergonómicos de bebé! Este proyecto tiene como objetivo presentar una página web atractiva y funcional para promover y vender zapatos ergonómicos diseñados específicamente para bebés.



## Características
Diseño moderno y atractivo que resalta la importancia de los zapatos ergonómicos para bebés.
Estructura de archivo organizada y legible.
Implementación de HTML semántico para una mejor accesibilidad y SEO.
Estilos CSS bien estructurados y modularizados.
Diseño responsivo para una óptima visualización en diferentes dispositivos.
Integración de imágenes de alta calidad para realzar la estética del sitio.



## Tecnologías utilizadas
HTML5
CSS3
